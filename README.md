# log

`log` is a Go logging library with GitHub Actions automation for pull request
validation and tagged GitHub releases.

## Release Automation

- Pull requests targeting `main` run `go test ./...` automatically in GitHub
  Actions.
- Releases are triggered by pushing a semantic version tag such as `v1.2.3`.
- The release workflow fetches the full git history, reruns repository tests,
  and publishes a GitHub release directly from GitHub Actions.
- Release notes are generated by GitHub from repository history so each
  published release includes changelog content scoped to that release.
- Rerunning the release workflow for a tag that already has a GitHub release
  skips publication to avoid duplicate releases.
